# Euclid

Initial exploration of the first Euclid data. Want to measure some depths, sizes, and get a feel for the improvements on the ground-based imaging.

## Depth codes

These codes relate to running depths and PSFs on the image. Modified from previous code.

## Cutout codes

Allow me to pass coordinates to quickly recover Euclid, ground-based, HST and JWST stamps (if they exist).

## File management

Getting the data into a usable stage, including mosaicing, renaming files, downloading files, cropping mosaics.

## Validation

Tests on the Euclid PV data to verify its science readiness.
